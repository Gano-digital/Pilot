With Fallback Chain (Python)
──────────────────────────────
def get_response(messages):
    try:
        # Try Claude Opus first
        return claude.chat.completions.create(...)
    except RateLimitError:
        # Fallback to V4-Pro
        return deepseek.chat.completions.create(
            model="deepseek-v4-pro",
            messages=messages
        )
    except Exception:
        # Fallback to V4-Flash
        return deepseek.chat.completions.create(
            model="deepseek-v4-flash",
            messages=messages
        )

  temperature: 0.0, top_p: 1.0, reasoning_effort: "none"

                            ERROR HANDLING
================================================================================

Common Errors
──────────────────────────────
401 Unauthorized     → Invalid API key
429 Too Many Requests → Rate limited (use exponential backoff)
500 Server Error     → DeepSeek service issue (fallback)
Timeout (10+ min)    → Inference took too long

Exponential Backoff Example
──────────────────────────────
for attempt in range(3):
    try:
        return client.chat.completions.create(...)
    except RateLimitError:
        wait_time = 2 ** attempt
        sleep(wait_time)  # 1s, 2s, 4s
    except APIError:
        raise
