## 1. Dimensiones de comparación

Las siguientes dimensiones son las más relevantes para el segmento objetivo de Gano Digital: pequeñas y medianas empresas colombianas que buscan WordPress de alto rendimiento con facturación local.

### 1.1 Precio y modelo comercial

| Subdimensión | Por qué importa |
|---|---|
| Precio de entrada (plan básico) | Define quién puede comprarlo y posiciona al proveedor en el mercado |
| Modelo de facturación (mensual / anual / biennial) | Impacta el flujo de caja del cliente; descuentos anuales son práctica común |
| Moneda de cobro (COP / USD) | Para negocios colombianos, cobrar en COP reduce fricción y riesgo cambiario |
| Método de pago local (PSE / Nequi / Daviplata / tarjeta) | Determinante de conversión en Colombia |
| Política de reembolso | Reduce barrera de compra; refuerza confianza |

### 1.2 Soporte

| Subdimensión | Por qué importa |
|---|---|
| Idioma del soporte (español CO / es general / inglés) | Calidad percibida y resolución efectiva de tickets |
| Canales (chat en vivo / ticket / teléfono / WhatsApp) | WhatsApp es canal preferido en LATAM; su ausencia es fricción |
| Horario declarado (24/7 / horas hábiles CO) | Hosting es 24/7 por naturaleza; soporte limitado es riesgo |
| Tiempo de respuesta prometido (first response SLA) | Indicador de calidad; se puede verificar con prueba manual |
| Soporte en WordPress (básico / gestionado / especializado) | Diferencia hosting genérico de managed WP real |

### 1.3 WordPress Managed (gestión incluida)

| Subdimensión | Por qué importa |
|---|---|
| Actualizaciones automáticas de WP core / plugins / temas | Reduce superficie de ataque y carga de mantenimiento |
| Staging / entorno de pruebas | Flujo profesional de despliegue sin riesgo |
| Backups automáticos (frecuencia / retención / restauración) | Seguro de datos; diferenciador crítico |
| Caché gestionado (servidor / plugin / CDN incluido) | Impacta Core Web Vitals directamente |
| Seguridad gestionada (WAF / escaneo de malware / SSL) | Demandado por clientes no técnicos |
| Migraciones WordPress incluidas | Reduce fricción de adquisición de clientes de otros proveedores |

### 1.4 Facturación y cumplimiento local (Colombia)

| Subdimensión | Por qué importa |
|---|---|
| Factura electrónica DIAN | Obligatorio para empresas colombianas; diferenciador vs. proveedores extranjeros |
| NIT registrado en Colombia | Indica presencia legal local; genera confianza y posibilita deducción de IVA |
| Soporte para retención en la fuente | Relevante para clientes persona jurídica |
| Contrato en español / legislación colombiana | Reduce riesgo legal; preferido por PYMES |
| Datos alojados en Colombia o LATAM (opción) | Relevante para DIAN, SIC y sensibilidad de datos |

### 1.5 Rendimiento declarado

| Subdimensión | Por qué importa |
|---|---|
| Uptime SLA declarado (%) | Referencia mínima esperada (99.9 % ≈ ~8.7 h downtime/año) |
| Tipo de infraestructura declarada (NVMe / SSD / HDD, shared / VPS / cloud) | Impacta velocidad y aislamiento de recursos |
| CDN incluido (sí / no / opcional, puntos de presencia LATAM) | Latencia para usuarios colombianos |
| Límite de recursos (inodos, bandwidth, procesos simultáneos) | "Ilimitado" frecuentemente tiene throttling oculto |
| Prueba de velocidad publicada (GTmetrix / PageSpeed / propio) | Autoevaluación; verificable externamente |

### 1.6 Diferenciadores estratégicos (contexto Gano Digital)

| Subdimensión | Por qué importa para Gano |
|---|---|
| Presencia en redes sociales / comunidad colombiana | Visibilidad orgánica y social proof local |
| Ecosistema / bundles (correo, dominio, sitio, tienda) | Propuesta de valor integrada vs. á la carte |
| Programa de afiliados o revendedores | Canal de crecimiento indirecto |
| Tiempo en el mercado colombiano | Establece confianza versus propuestas nuevas |
| Enfoque WooCommerce / e-commerce | Relevante para clientes de tiendas en línea |

## 4. Guía de investigación sugerida

### 4.1 Proceso de relevamiento (pasos)

1. **Identifica competidores candidatos** — empezar por los que aparecen en búsquedas Google Colombia: "hosting WordPress Colombia", "hosting wordpress colombia precio".
2. **Registra la URL de precios y la fecha exacta** — los precios cambian; una captura de pantalla o copia del texto es la fuente más confiable.
3. **Haz una prueba de soporte (opcional pero valioso)** — abre un chat de ventas y pregunta en español; mide tiempo de respuesta y calidad.
4. **Verifica facturación** — busca en el pie de página si tienen NIT colombiano; busca si emiten factura DIAN.
5. **Mide velocidad de un sitio demo** — si el proveedor tiene demo o sitios de clientes públicos, corre GTmetrix desde Bogotá (si está disponible) o desde LATAM.
6. **Anota gaps** — anota qué no pudiste encontrar públicamente; algunos datos solo se obtienen con una compra de prueba.

### 4.2 Categorías de proveedores a considerar (sin nombres hasta investigación)

Para guiar la búsqueda, se sugiere investigar estas categorías de jugadores:

| Categoría | Descripción |
|---|---|
| **Hosting local colombiano** | Proveedores con sede y facturación en Colombia; probable fortaleza en DIAN / COP |
| **Hosting regional LATAM** | Sedes en MX, AR, BR con presencia en CO; mayor escala, menor localización |
| **Hosting global con operación LATAM** | Proveedores internacionales con datacenter en LATAM o soporte en español |
| **Agencias de WordPress Colombia** | No venden hosting directo, pero compiten por la relación con el cliente WP |
| **Revendedores GoDaddy / similares** | Competencia indirecta en el mismo programa reseller |

### 4.3 Señales de alerta al evaluar un competidor

- Precios en USD sin opción COP → barrera para PYMES colombianas (ventaja Gano)
- Sin factura DIAN → fricción contable para personas jurídicas (ventaja Gano en Fase 4)
- Soporte solo en inglés → fricción de servicio (ventaja Gano)
- Uptime "ilimitado" o "100%" sin SLA escrito → promesa no auditable
- "Managed WordPress" sin especificar qué incluye → probable shared hosting renombrado
