╔═══════════════════════════════════════════════════════════════════════════╗
║                    ❌ PROBLEMAS COMUNES (SOLUCIONES RÁPIDAS)             ║
╚═══════════════════════════════════════════════════════════════════════════╝

"Plugin no aparece en WordPress"
  → Verifica: /wp-content/plugins/gano-content-importer/gano-content-importer.php
  → Recarga página (F5)

"Páginas no se crean"
  → Verifica: /wp-content/debug.log (WordPress debug log)
  → Desactiva otros plugins que interfieran

"CSS y JS no cargan (404)"
  → Copia nuevamente los archivos
  → Limpia caché si usas cache plugin
  → Verifica permisos (chmod 644)

"Se ve sin estilos"
  → Verifica: gano-child theme está ACTIVO
  → Copia gano-sota-animations.css nuevamente
  → Limpia caché navegador (Ctrl+Shift+Delete)

"Las animaciones no funcionan"
  → Verifica: scroll-reveal.js existe en /js/
  → DevTools Console → ¿Dice "Scroll reveal initialized"?
  → Desactiva plugins de optimización que minifiquen JS
