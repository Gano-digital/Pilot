1. INSTALAR EN STAGING
   ├─ Copiar plugin a wp-content/plugins/
   ├─ Copiar gano-sota-animations.css a tema gano-child/
   └─ Verificar permisos de archivo (644 archivos, 755 directorios)

2. ACTIVAR PLUGIN
   ├─ WordPress Admin → Plugins
   ├─ Buscar "Gano Digital Content Hub Importer v2.0"
   ├─ Click "Activate"
   └─ Verificar wp-content/debug.log

3. REVISAR PÁGINAS CREADAS
   ├─ Admin → Pages (deberías ver ~20 nuevas)
   ├─ Estado: DRAFT (listo para edición)
   ├─ Todas tienen featured image placeholders
   └─ Template: Elementor Canvas

4. PROBAR EN FRONTEND
   ├─ Abrir una página en navegador
   ├─ Verificar animaciones:
   │  ├─ h1 slide-in desde arriba
   │  ├─ Bullet points reveal staggered
   │  └─ CTA button ripple effect
   └─ Prueba en móvil (<480px)

5. VALIDAR RESPONSIVIDAD
   ├─ Chrome DevTools → Toggle device toolbar
   ├─ Probar 480px (mobile), 768px (tablet), desktop
   ├─ Verificar que layout es clean en cada breakpoint
   └─ Scroll reveal debe funcionar al scroll

6. PERSONALIZAR CTAs (Opcional)
   ├─ Decidir si variar URLs por categoría
   ├─ Si sí, usar filter hook (ver README.md)
   ├─ Si no, dejar como está (/contacto genérico)
   └─ Editar cada página y cambiar enlaces si quieres

7. PREPARAR PARA PRODUCCIÓN
   ├─ Subir imágenes featured a /uploads/2026/03/
   ├─ Revisar copy de cada página (está en español latino)
   ├─ Publicar páginas (cambiar status de DRAFT a PUBLISH)
   └─ Actualizar menú de navegación si quieres incluirlas

8. ELIMINAR PLUGIN (Una vez terminado)
   ├─ WordPress Admin → Plugins → Deactivate
   ├─ Las páginas PERMANECEN
   ├─ Click "Delete" si quieres limpiar
   └─ El plugin no se necesita después de instalación
