# Gano Digital — Project Brief

## Vision
Construir la plataforma de hosting WordPress #1 para el mercado colombiano.
Modelo: Reseller de alta gama con branding "Soberania Digital".

## Target
- Agencias digitales colombianas
- Emprendedores tech-savvy
- Empresas que necesitan hosting WordPress premium en COP

## Core Features
1. Hosting WordPress gestionado (reseller)
2. Checkout y cobro al cliente en COP vía **GoDaddy Reseller** (alineado a `TASKS.md`)
3. Soporte en espanol (FreeScout — roadmap)
4. Facturacion DIAN (roadmap)
5. Panel WHMCS (roadmap Fase 4)

## Diferenciadores
- Precios en COP, no USD
- Soporte local colombiano
- Infraestructura NVMe + CDN
- Tono "Manifiesto Tecnico" (premium, no barato)
- Zero-Trust security
