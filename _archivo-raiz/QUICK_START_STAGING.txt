───────────────────────────────────────────────────────────────────────────────
 ❌ TROUBLESHOOTING
───────────────────────────────────────────────────────────────────────────────

PROBLEMA: "Plugin no aparece en lista"
  → Verifica que gano-content-importer.php esté en la carpeta correcta
  → Recarga la página de plugins

PROBLEMA: "Páginas no se crean"
  → Revisa /wp-content/debug.log
  → Verifica PHP 7.4+ instalado
  → Verifica WordPress 5.9+ instalado

PROBLEMA: "Animaciones no funcionan"
  → Verifica que CSS esté en: wp-content/themes/gano-child/
  → Verifica que JS esté en: wp-content/plugins/gano-content-importer/js/
  → Limpia caché del navegador (Ctrl+Shift+Delete)
  → Desactiva plugins que puedan interferir (Elementor cache, etc)

PROBLEMA: "Las páginas se ven feas"
  → Copia gano-sota-animations.css nuevamente
  → Verifica que el child theme esté activo
  → Limpia caché (si usas caché plugin)
